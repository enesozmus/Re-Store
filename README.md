# Re-Store
> *  ['http://localhost:5000/api/Products'] ve ['http://localhost:5000/api/Basket'] url'lerini tüketen ilk React uygulamam
> *  Tamamlanma Oranı: %35

## İzlence
> * Ürünler listelendi.
> * Ürün detay sayfası oluşturuldu.
> * Ürün ekleme ve çıkarma fonksiyonlarıyla birlikte sepet eklendi.
<br/>
